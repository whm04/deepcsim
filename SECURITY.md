## Supported Versions

Only the **latest released version** of DeepCSim is actively supported with security updates.

Users are strongly encouraged to:
- Keep DeepCSim up to date
- Regularly update dependencies
- Run automated tests when upgrading to new versions

Staying up to date ensures you benefit from the latest **features, bug fixes, and security patches**.

## Reporting a Vulnerability

If you think you found a vulnerability, and even if you are not sure about it, please report it right away by sending an email to: **mzaliwahib04@gmail.com**.
 
Please try to be as explicit as possible, describing all the steps and example code to reproduce the security issue.

I will review the report carefully and respond as quickly as possible.
